# BTC vs. S&P 500 Volatility Analysis

This repository contains a Python script for analyzing and comparing the 30-day rolling volatility of Bitcoin (BTC) and the S&P 500 index over the period from 2013 to 2023. The script uses historical financial data fetched from Yahoo Finance and visualizes the volatility trends of these two assets over time.

---

## How to Use

### 1. Clone the Repository
```bash
git clone https://github.com/<YourUsername>/<YourRepositoryName>.git
cd <YourRepositoryName>
```

### 2. Install Dependencies
Ensure you have Python installed and the required libraries. Install the libraries using `pip`:
```bash
pip install pandas matplotlib yfinance
```

### 3. Run the Script
Execute the script to generate the volatility comparison plot:
```bash
python volatility_analysis.py
```

---

## Required Libraries
The script requires the following Python libraries:
- **pandas**: For handling time-series data and calculating daily returns.
- **matplotlib**: For creating the volatility plot.
- **yfinance**: For fetching historical financial data from Yahoo Finance.

To install all required libraries, run:
```bash
pip install pandas matplotlib yfinance
```
